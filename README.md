🎬 Show Tracker API

A REST API built with FastAPI that lets users track shows they've watched, want to watch, watching, and rate/review them. Includes JWT authentication.

🚀 Tech Stack

FastAPI — web framework

SQLAlchemy — ORM

Pydantic — data validation

JWT and pwdlib - password hashing & JWT auth 


⚙️ Setup & Installation
1. Clone the repo
git clone https://github.com/VishallMan01/Show_Tracker-.git
cd show-tracker
3. Create a virtual environment
bashpython -m venv venv
source venv/bin/activate
4. Install dependencies
pip install -r requirements.txt
5. Set up environment variables
bashcp .env.example .env
Edit .env with your secret key
6. Start the server
fastapi dev 
7. View API docs
Open http://localhost:8000/docs in your browser (Swagger UI — auto-generated by FastAPI!)
